# Module 2: Introduction to HTML and CSS

## HTML (Hypertext Markup Language) 

* Describe the purpose of HTML in a web browser.
* Demonstrate the use of HTML tags, including how to ensure that you use the correct syntax.
* Identify how HTML elements are utilized in a web document.

## HTML documents

* Basics of an HTML page, including the root, head, body and other elements.
* How to build basic HTML documents by combining common key elements and tags in order to create a structure.
* How to create a link between HTML documents with the use of an href or hypertext reference attribute to redirect users between the documents.
* Adding images to a web page with HTML code
* Utilizing HTML to manipulate data in tables
* Building HTML forms with a wide range of input types.

## Document Object Model (DOM)

* A tree structure or model of objects in an HTML page.
* The DOM is created from an HTML page by exploring the element that generate it and visualizing it in diagram form.

Common instances in which web developers interact with the DOM through JavaScript includes:

* Updating content.
* Setting up events.
* Animating HTML elements.

## Web accessibility

* How people with disabilities can understand, navigate, and interact with websites.
* Assistive technologies used to browse the web like screen reading and speech recognition software.
* Basic techniques to improve the accessibility of your web projects, such as correct HTML structure.

## Basics of CSS (Cascading Style Sheets)

* Writing CSS rules in order to style elements within HTML document.
* The CSS box model and its key features.
* The concept of document flow.
