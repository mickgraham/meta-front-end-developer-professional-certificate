# Module 4: Introduction to Web Development

## Knowledge gained:

* Describe the web developer job role.
* Distinguish between front and backend and full stack developers.
* Explain how data moves through the Internet.
* Describe the technologies that underpin the Internet.

## Lab skills:

* Ability to use HTML to create a simple web page.
* Use CSS to define the style of a simple web page.
* Outline the concepts that exist in most UI frameworks using Bootstrap.
* Use the Bootstrap CSS framework to create web pages.

## Follow up:

* Use Bootstrap documentation to reproduce and modify CSS components.
* Use Bootstrap themes to speed up your workflow.
* Describe the basics of React in relation to other frameworks and web technologies.
