# Module 2: Command Line

This module focuses on the command line and its functionalities in Linux.

## Command Line Basics

* You learned how to execute commands in Linux using the command line.
* Key commands for traversing, creating, renaming, and deleting files were introduced.

## File Management Techniques

* You explored how to use piping and redirection to automate workflows.
* The PWD command was used to determine the current working directory.

## Directory and File Operations

* You can now create and navigate directories and files, including creating two directories: dir 1 and dir 2.
* You learned to use Grep for searching files, folders, and file contents.
