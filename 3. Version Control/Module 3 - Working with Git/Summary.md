# Module 3: Working with Git

This course content focuses on the key concepts and skills related to using Git and GitHub for version control.

## Understanding Git and GitHub

* Learn the principles of Git and how to utilize a GitHub repository, including branching and merging code.
* Differentiate between remote and local repositories in GitHub and understand the Git workflow.

## Git Commands and Workflow

* Perform essential Git commands such as add, commit, push, and pull to manage code revisions.
* Use branches to maintain a clear and stable workflow, and understand how HEAD identifies the current branch.

## Advanced Features and Collaboration

* Compare changes across files and commits using diff commands, and identify authors of changes with blame.
* Create repositories through forking, enhancing collaboration in software development projects.
