# Module 2: Interactive CSS

## Module Summary

This course module focuses on interactive CSS, enhancing your skills in styling web elements.

### Creating Layouts

* You learned about CSS web layouts, including grid and flexbox modules, and common layout examples like column grids and stacks.
* You applied this knowledge by creating a custom layout using grids.

### CSS Selectors

* The module covered advanced CSS selectors, their specificity, and combination selectors for granular selection of elements.
* You also explored pseudo-classes and pseudo-elements, along with their practical applications.

### CSS Effects and Debugging

* You discovered how to enhance web presentation using CSS effects, including text effects and basic animations.
* The debugging section taught you to inspect and resolve CSS and HTML issues, utilizing browser developer tools for effective troubleshooting.
