# Module 1: Introduction to UX and UI Design

This course content provides an overview of the key concepts and skills learned in the module on UX and UI design.

### Introduction to UX and UI

* Differentiation between UX (User Experience) and UI (User Interface) and identification of career options in these fields.
* Understanding of UX goals and usability quality components, including learnability, efficiency, memorability, error, and satisfaction.

### UX Process Overview

* Recognition of the key stages in the UX process: empathize, define, ideate, iterate, and test.
* Application of user-centered design principles through customer interviews and observations, including the creation of customer journey maps and personas.

### Introduction to UI and Design Tools

* Explanation of UI concepts, its history, and various types of design such as interaction design and human-centered design.
* Introduction to Figma as a design tool for creating wireframes and prototypes, emphasizing the importance of user-centered design in the redesign process.
