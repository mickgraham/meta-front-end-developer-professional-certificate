# Course Recap for Principles of UX and UI Design

This course provides an overview of the principles of UX and UI design, focusing on key concepts and practical applications.

## Introduction to UX and UI Design

* Differentiation between UX (User Experience) and UI (User Interface) design.
* Exploration of user-centered design, empathy tools, and personas in UX.

## Evaluating Interactive Design

* Examination of evaluation methodologies and accessibility considerations.
* Study of best practices in navigation and form design through practical examples.

## Applied Design Fundamentals

* Fundamentals of using Figma for design, including wireframing and prototyping.
* Overview of iterative design principles and usability testing techniques.

## Designing Your UI

* Techniques for enhancing designs and creating high-fidelity component-based designs.
* Guidance on using mood boards and creating design systems in Figma.

## Final Assessment

* Application of learned skills through a practical assignment on the Little Lemon website.
* Preparation for a graded assessment to evaluate understanding of course content.
