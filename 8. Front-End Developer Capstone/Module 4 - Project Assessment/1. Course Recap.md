# Module 4 - Course Recap: Capstone Project

This course content focuses on the final module of the Front-End Developer Capstone project, summarizing the key learning experiences throughout the course.

## Project Overview

* The course began with an introduction to the syllabus, project setup, and creating a React app while tracking changes with Git.
* In the second module, students worked on UX/UI planning, wireframes, and applied design fundamentals using Figma.

## Project Functionality

* The third module involved coding a table reservation system for the Little Lemon web app, implementing unit testing, and working with an API.
* Students polished the UI, validated user data, and wrote additional tests for their application.

## Assessment and Collaboration

* The current module emphasizes project assessment, including a self-review and peer review of another learner's solution focusing on usability, accessibility, design, and code.
* Students will compare their work with peers and completed solutions to enhance their understanding and skills as React developers.

The module concludes with a graded assessment quiz and a course wrap-up, allowing students to reflect on their learning journey and achievements.
